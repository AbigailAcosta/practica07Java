# Practica07 en NetBeans - Java
- **Registrar:** Un progarma en Java para registrar nombres de productos en un _ComboBox_, el cual es desplegable, lo que permite visualizar los productos ingresados.

- **Saludar:** Un programa donde se debe ingresar nombre, luego al dar click en un botón llamado "¡Saludar!" genera una ventanita con un saludo y el nombre ingresado.
